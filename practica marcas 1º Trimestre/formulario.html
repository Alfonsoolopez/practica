Para este formulario, he empleado las siguientes etiquetas y atributos.

<fieldset>
Se emplea para agrupar de forma lógica varios campos de un formulario.
<form>
Se emplea para insertar un formulario en la página.
<legend>
Nos indica el nombre de un apartado del formulario.
<label>
Se emplea para definir el título o leyenda de los campos de un formulario.
<strong>
Pone en negrita el contenido
<input>
Se emplea para insertar un control en un formulario. Dentro de esta etiqueta hemosutilizado los siguientes atributos:
fille
Indica que tenemos que insertar un contenido
date
Especifica la fecha y la hora en la que se realizó el cambio
radio
Permite indicar qué opción preselecciona
value
El valor que se envía cuando el usuario elige esa opción
maxlength
Máximo numero de caracteres para los controles de texto
